A set of Peer nodes , or, simply, peers. Peers are a fundamental element of the network because they host ledgers and smart contracts. Recall that a ledger immutably records all the transactions generated by smart contracts (or chaincode). Smart contracts and ledgers are used to encapsulate the shared processes and shared information in a network, respectively. 1. The Fabric ledger is maintained by each peer and includes the blockchain and worldstate 1. A separate ledger is maintained for each channel the peer joins 1.
